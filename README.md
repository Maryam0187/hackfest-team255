# hackfest-team255

### Prerequisite

1. Downlaod docker desktop app and docker is running
2. Go to docker folder and run **docker-compose up**

## To Run Backend

1. First run npm install
2. Run **npm run db** (run migrations and seed)
3. Run **npm run server** (backend running)

## To Run Frontend
1. run yarn install
2. run yanr start
